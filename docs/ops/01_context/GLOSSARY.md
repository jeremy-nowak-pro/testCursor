# Glossary

- **MVP**: Minimum Viable Product
- **DoD**: Definition of Done
- **ADR**: Architecture Decision Record
- **Task ID**: Identifiant unique d'un lot livrable (ex. `CAT-001`), voir `docs/ops/04_tasks/TASK_IDS.md`
- **Agent Backend**: Travail Laravel (routes, modeles, tests Feature cote serveur)
- **Agent Frontend**: Travail Inertia/React (pages, composants, pas de logique metier serveur)
- **develop**: Branche d'integration; toutes les PR features y fusionnent avant `main`
