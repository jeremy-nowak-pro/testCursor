# Sprint 01

## Goal
Etablir la base technique du MVP et livrer un premier parcours utilisateur complet:
home -> categorie -> filtres -> detail produit + auth.

## Tasks
- Finaliser conventions projet et prerequis de setup
- Initialiser stack Laravel + Inertia React
- Mettre en place auth Laravel (register/login/logout)
- Implementer layout global (header categories + footer legal/cookies)
- Implementer page home et navigation vers categories
- Implementer listing categorie avec filtres (prix/type/format)
- Implementer page detail produit
- Definir et valider contrats des endpoints MVP
- Ajouter tests feature backend (auth, listing, detail)
- Ajouter tests frontend sur parcours principal
- Verifier lint/test/build en local

## Definition of Done
- [ ] Parcours principal fonctionnel en local sans blocage
- [ ] Auth operationnelle
- [ ] Contrats listing/detail documentes et respectes
- [ ] Tests prioritaires verts
- [ ] Aucun blocant QA ouvert pour Sprint 01
