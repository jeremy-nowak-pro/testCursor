# Tooling Checklist

- [ ] Laravel Pint configured and runnable
- [ ] ESLint configured for React/Inertia code
- [ ] Prettier configured and non-conflicting with ESLint
- [ ] Backend tests configured (PHPUnit or Pest)
- [ ] Feature tests for auth and catalog routes added
- [ ] Frontend baseline tests configured
- [ ] CI pipeline configured for lint + tests + build
- [ ] Pre-commit checks configured (format/lint quick checks)
- [ ] Error logging strategy documented
- [ ] Environment setup documented for new contributors
